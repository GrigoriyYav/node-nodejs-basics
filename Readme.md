## File system (src/fs)

```bash
npm run fs:copy
npm run fs:create
npm run fs:delete
npm run fs:list
npm run fs:read
npm run fs:rename
```

## Command line interface(src/cli)

```bash
npm run cli:args -- --addYourArgHere yourArgValue --Param2 value2
# for bash
RSS_1=Hello RSS_2=World npm run cli:env
# for Windows PowerShell
$env:RSS_1="Hello"; $env:RSS_2="World"; npm run cli:env
```
